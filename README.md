## 📦 Project Structure

```
complete-try/
├── backend/       # Node.js + Express API + MongoDB
└── frontend/      # React + Vite Frontend
```


### 🔧 Backend Setup

```bash
cd backend
npm install
npm start
```

### 🔧 Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
